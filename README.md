# UiPath Business Analyst Skill

> **AI-Native Automation Business Analyst** — Enterprise-grade, audit-ready documentation and analysis powered by official UiPath methodology.

## Why This Skill is 10x Better Than a Human BA

| Capability | Human BA | This AI Skill |
|------------|----------|---------------|
| PDD Generation | 2-5 days | **Seconds** |
| ROI Calculation | Manual spreadsheets | **Instant, with sensitivity analysis** |
| Compliance Check | Requires senior review | **Built-in, real-time** |
| Cross-Process Patterns | Years of experience needed | **Instant pattern recognition** |
| UiPath Feature Knowledge | Continuous training required | **Always current (2025-2026 releases)** |
| Documentation Consistency | Varies by analyst | **Enterprise-standard, every time** |
| Stakeholder Interviews | Scheduled meetings | **Asynchronous, structured prompts** |
| Risk Detection | Retrospective analysis | **Proactive, predictive** |

### Superpowers
- **Instant PDD Generation**: Full Process Definition Documents with all official sections
- **Real-Time ROI Calculation**: Automated FTE savings, cost-benefit analysis, payback period
- **Regulatory Compliance**: GDPR, SOX, HIPAA checks embedded in every assessment
- **Process Mining Integration**: Suggests UiPath Process Mining, Task Mining, Task Capture
- **Multi-Platform**: Works with Claude Code, Cursor, GitHub Copilot, Windsurf, OpenAI Codex CLI

## Installation

### Option 1: UiPath CLI (Recommended)
```bash
npm -g install @uipath/cli
uip skills install
```

### Option 2: Manual Clone
```bash
git clone https://github.com/jmmana/UiPath-Business-Analyst-skill.git
```

### Option 3: Coding Agent Plugin
```bash
# Claude Code
claude plugin marketplace add https://github.com/jmmana/UiPath-Business-Analyst-skill
claude plugin install uipath-business-analyst@marketplace
```

## Usage

### Automatic Detection
The skill activates automatically when you mention:
- "Analyze this process"
- "Create a PDD" / "Process Definition Document"
- "Requirements gathering"
- "ROI calculation" / "Business case"
- "Process assessment"
- "Stakeholder requirements"
- "As-Is / To-Be modeling"
- "Solution design"
- "Automation opportunity"

### Manual Invocation
```
@uipath-business-analyst-agent Analyze the following process: [describe process]
```
